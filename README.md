## Ejecutar localmente

Clonar proyecto

```bash
  git clone https://github.com/kioshiokamoto/pets-web
```

Ir a ruta de proyecto

```bash
  cd pets-web
```

Instalar dependencias

```bash
  yarn
```

Iniciar servidor

```bash
  yarn dev
```
